Release Notes

Targeted refinements to measurements, media handling, pagination and memo workflows.

March 2026 — Release 3
01
📏
Enhanced

Consistent Measurements Formatting

Measurements across the application are now consistently displayed with two decimal places for clearer, more professional documentation.

  • All measurement fields now follow a standard format with two decimal places, e.g. 10.01 × 6.85 × 0.00
  • Ensures consistency across inventory views, certificates, and other measurement displays
Standardized formatting helps avoid confusion and supports clearer communication with customers and partners.
02
📷
New

HEIC Format Support for Certificates & Media

Easily upload modern iPhone-style images without manual conversion.

  • Support added for HEIC image files when uploading certificate images and media assets
  • Reduces the need to convert HEIC files to JPEG/PNG before upload
03
🔄
Improved

Consistent Pagination Across Tables

Pagination state is now preserved when navigating between list and detail views, so you always return to the same page you were working on.

  • Resolved an issue where returning from a detail view reset table pagination back to page 1
  • Applied consistently across all modules that use paginated tables
04
🖹
Enhanced

Measurements on Barcode Labels

Barcode label PDFs now use the same standardized measurements format as the rest of the application.

  • Updated barcode label PDF generation to use the new two-decimal measurements formatting
  • Ensures on-label measurements match what you see in the system
05
👥
Improved

Supplier Dropdown Company Names in Memo In

Supplier selection in Memo In is now clearer and easier to use.

  • Updated the supplier dropdown in Memo In to display the supplier’s company name
  • Makes it easier to identify the correct supplier when multiple contacts share similar personal names
