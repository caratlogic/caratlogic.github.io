Memo In & Memo Out Management

End-to-end memo transaction lifecycle — from issuance to returns — now fully managed within the platform.

  • Issue Memo Out consignments and process returns with auto-generated PDF documentation
  • Record Memo-In receipts to maintain a complete inbound consignment ledger
  • Initiate and track Memo-In returns with full audit trail support

Enhanced

Barcode Label Customization

Greater flexibility and precision when designing barcode labels for your inventory.

  • Full control over font size and style for both label headings and data values on printed barcodes
  • Introducing Pin Value — lock a specific attribute value to persist across an entire row, ensuring consistency in batch label printing

Dedicated Gems & Diamond Modules

A refined, role-based experience that surfaces only what matters to your business vertical.

  • Gems users access a dedicated workspace with gem-specific features, settings, and navigation
  • Diamond users access a dedicated workspace with diamond-specific features, settings, and navigation
The interface now adapts to each user's inventory category — reducing clutter, minimizing errors, and delivering a more intuitive day-to-day experience.
